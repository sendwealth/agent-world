# Generated Protobuf Code

This directory contains code generated from `protocol/*.proto` by `make proto`.

**When to regenerate:**
- After modifying `protocol/a2a.proto`
- After upgrading `grpcio-tools` or `protobuf` versions

**How to regenerate:**
```sh
make proto
```

The Python output (`gen/python/a2a_pb2.py`, `gen/python/a2a_pb2_grpc.py`) is
committed to the repo so that the codebase is importable without a build step.
Rust output (`gen/rust/`) is gitignored and regenerated by `cargo build`.
