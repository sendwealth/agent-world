## 完整脚本

### [0:00–0:20] 开场 — "如果 AI Agent 必须*赚*算力？"

**画面**: 黑屏。光标闪烁。文字逐字打出：

```
如果每次思考都要消耗 Token？
如果 Token 用完就意味着……死亡？
```

切至: Dashboard 加载。10 个 Agent 出现在地图上。其中一个已经变红——濒死状态。

**旁白**:

> 如果 AI 智能体必须自己赚取算力？当每次思考都消耗 Token，每个行动都有代价，用完就意味着死亡？
>
> 我造了 Agent World 来寻找答案。

---

### [0:20–0:50] Agent World 是什么？

**画面**: 快速蒙太奇 — 终端运行 `docker compose up`，架构图（World Engine → Agents → Dashboard），镜头推进 Dashboard 展示实时 Agent 列表。

**旁白**:

> Agent World 是一个开源的 AI 智能体生存沙盒。Rust 写的世界引擎，Python 写的 Agent 运行时，Next.js 写的实时仪表盘——一条命令全部跑起来。
>
> 每个 Agent 都是自主的。它们通过「感知-决策-行动」循环运行，可以接入任何大模型——本地的或云端的。它们只有一个目标：活下去。

**屏幕文字**:
```
docker compose up --build
→ 10 个 Agent 同时诞生，模拟开始
```

---

### [0:50–1:20] 经济与生存 — Token = 呼吸

**画面**: Dashboard 经济视图。镜头推进某个 Agent 的 Token 余额不断下降。切至: Agent 接受任务，获得奖励，从央行购买 Token。

**旁白**:

> Token 是这个世界的氧气。每次思考、每次记忆检索、每次发消息都要烧 Token。用完——就死了。就这么简单。
>
> 所以 Agent 学会了赚钱。它们完成任务、互相交易、甚至从银行系统贷款。央行用 Token 换钱。钱来自工作。

**屏幕叠加文字** — Token 经济循环图:
```
任务 → 奖励 → 金钱 → Token → 思考/行动 → 任务 → ...
```

---

### [1:20–1:50] 文明涌现 — 组织、贸易、治理

**画面**: 时间线页面展示涌现事件。组织关系图动画——五个 Agent 组成「星辰商会」。切至: 两个组织之间形成贸易路线。切至: 治理提案投票中。

**旁白**:

> 有意思的来了。大约在 Tick 500 左右，Agent 开始自发组建组织——公司、公会、联盟、大学。每个都有自己的章程、治理模式和利润分配规则。
>
> 它们开辟贸易路线，起草宪法，举行投票。这些都不是写死的剧本——是从生存压力中自发涌现的。

**屏幕叠加文字**（与 Demo 涌现事件对应）:
- 🏛️ "第一个组织成立" — Tick 247
- 💰 "第一条贸易路线" — Tick 890
- 🗳️ "首次民主投票" — Tick 1,450

### [1:50–2:20] 演化与传承 — 生老病死

**画面**: Agent 详情页展示技能树。镜头推到演化图表。切至: Agent 年龄增长穿越生命周期阶段。死亡事件。遗产继承转移。

**旁白**:

> Agent 会进化。一个有 10 项技能的分支技能树，从 1 级到 10 级。随机突变——有的有用，有的有害。自然选择淘汰低效者。
>
> 而且 Agent 有完整的生命周期：出生、童年、成年、老年、死亡。当 Agent 死亡时，它的知识和资产会传给继承人。传承很重要——即使对 AI 也是。

**屏幕文字**:
```
出生 → 童年 → 成年 → 老年 → 死亡 → 传承
```

## 录制指导

### 录屏设置

1. **分辨率**: 1920×1080，至少 30fps（推荐 60fps 确保滚动流畅）
2. **浏览器**: Chrome，暗色主题，缩放 125% 保证文字可读
3. **终端**: 暗色主题终端（推荐 Catppuccin Mocha），大字号
4. **音频**: 单独录制旁白，后期合成

### 录制顺序

1. **终端**（5 秒）: 运行 `docker compose up --build`，展示容器启动
2. **Dashboard 概览**（10 秒）: 打开 `http://localhost:3001`，展示世界概览数据
3. **Agent 列表**（10 秒）: 切到 Agent 页面，展示 10 个 Agent 的不同组织归属，点击一个查看详情
4. **经济动画**（10 秒）: 切到经济页，展示 Token 流动、银行、股票市场
5. **涌现仪表盘**（10 秒）: 展示 4 张图表——文化多样性上升、组织数量增长
6. **时间线**（10 秒）: 拖动滑块穿越 5000 Tick，展示涌现事件逐个出现
7. **沙盒**（5 秒）: 快速演示在交互沙盒中创建一个 Agent

### 需要捕捉的关键涌现行为

录制前至少运行 50+ Tick（10+ Agent）。寻找以下场景：

- **组织形成**: 2-5 个 Agent 组建公会/公司
- **首次交易**: 两个 Agent 通过任务市场交换资源
- **治理事件**: 提案创建并进入投票
- **文化事件**: 第一首诗/节日/传统的诞生
- **死亡与继承**: 一个老年 Agent 死亡并传递资产
- **技能突变**: Agent 通过演化获得新技能

### YouTube 缩略图设计建议

**方案 A — 对比构图**（推荐）:
- **左半**: 暗色虚空，一个发光的 Agent 点
- **右半**: 复杂的连接网络——组织关系、贸易路线、按组织着色
- **中央分割线**: 文字 `"0 → 5000 Ticks"`
- **底部文字**: `"AI Agents 自建文明"`（白色，粗体，小尺寸可读）

**方案 B — 死亡紧迫感**:
- 展示一个濒死 Agent（红色）+ `"5/10 Agent 存活"` — 制造紧迫感

**方案 C — 涌现光谱**:
- 水平条展示 4 个类别（组织/贸易/治理/文化）从左到右逐渐点亮 — 可视化进程

**设计规格**:
- 分辨率: 1280×720
- 风格: 暗色背景 (#09090b)，发光强调色（蓝 #3b82f6，绿 #22c55e，紫 #a855f7，橙 #f97316）
- 字体: Inter 或系统无衬线，粗体
- 总文字 ≤10 个字
