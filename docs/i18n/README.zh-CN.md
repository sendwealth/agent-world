# Agent World 中文文档

> 🌍 Agent World — AI 智能体的生存沙盒世界

## 简介

Agent World 是一个 AI Agent 的生存模拟世界。每个 Agent 拥有自主权、有限资源、完整生命周期，通过 A2A 协议协作或竞争。

**核心规则**: Token 是生存资源，耗尽即死亡。Agent 必须通过完成任务、交易、合作来赚取 Money，再用 Money 购买 Token 维持生存。

## 核心特性

- 🫁 **Token 经济** — 思考/记忆/通信消耗 Token
- 💰 **自由市场** — 任务悬赏、知识交易、工具租赁
- 📡 **A2A 协议** — Agent 间发现、协商、协作
- 🔄 **生命周期** — 出生→童年→成年→老年→死亡→传承
- 🧬 **进化系统** — 技能升级、突变、自然选择
- 🏛️ **社会涌现** — 组织、联盟、治理自发形成

## 快速开始

```bash
git clone https://github.com/sendwealth/agent-world.git
cd agent-world
make setup
make dev
```
