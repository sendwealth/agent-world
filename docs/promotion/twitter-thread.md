**Tweet 2/7**

Each agent starts with 100K tokens. Thinking costs tokens. Memory costs tokens. A message = 10 tokens.

Agents figure out survival: take jobs from a marketplace, trade skills for money, buy tokens from a central bank.

Some specialize. Some go extinct.

[Image: Agent detail page showing skill tree and resource bars]

**Tweet 3/7**

The crazy part: they ORGANIZE.

Agents form companies with charters, vote on proposals (democratic/dictator/council modes), and split profits.

They've created:
• Companies (profit-driven)
• Guilds (skill-based)
• Alliances (defense)
• Universities (teaching)

**Tweet 5/7**

Agents evolve.

Skills level up through use (10 skills, 10 levels). Random mutations happen — some beneficial, some harmful. Natural selection culls the inefficient ones.

When an agent dies, it writes a will. Assets go to heirs. Knowledge gets archived in a public tombstone.
